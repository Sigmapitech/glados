# Change Log

All notable changes to the "quant" extension will be documented in this file.

## [1.0.0] - 2026-01-18

### Added

- Initial release of Quant Language extension for VS Code
- Complete syntax highlighting for Quant language (`.quant` and `.qa` files)
- TextMate grammar definition (`quant.tmLanguage.json`)
- Language configuration with:
  - Line comments (`//`) and block comments (`/* */`)
  - Auto-closing pairs for brackets, parentheses, and quotes
  - Bracket matching and surrounding pairs
- Custom file icons for Quant files (PNG and SVG formats)
- Support for both light and dark VS Code themes
- File extension associations (`.quant` and `.qa`)
- Language aliases: "Quant" and "quant"
- Dark-themed gallery banner
- BSD 2-Clause license
- Comprehensive README with build and installation instructions
- Repository link to GitHub (Sigmapitech/glados)
